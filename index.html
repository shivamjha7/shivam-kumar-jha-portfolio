Projects & Experience

Banking Application

Tech Stack: Java, Spring Boot, Hibernate, Spring Data JPA, MySQL, REST API, Postman

This is a backend-focused Banking Application developed using Java and Spring Boot, designed to perform core banking functionalities through a set of well-structured RESTful APIs. The application accepts and returns data in JSON format and is tested using Postman.

Key Features:

  • Create Account: Users can create a new bank account with initial details.
  • Retrieve Account: Fetch account details using account number or user ID.
  • Update Account: Modify existing account details such as name or contact info.
  • Delete Account: Permanently delete an account from the database.
  • Deposit & Withdraw: Perform deposit and withdrawal operations with validations.
  • Database Integration: Hibernate and JPA used for mapping Java objects to a MySQL relational database.
  • Error Handling: Custom exception handling and proper HTTP status codes for all API responses.

The project follows a layered architecture (Controller → Service → Repository), demonstrating my understanding of REST principles, backend development, and data management. This application serves as a strong foundation for building secure, scalable, and testable backend systems.

This project helped reinforce my frontend development skills and ability to convert real-world platforms into static responsive websites using core web technologies.
